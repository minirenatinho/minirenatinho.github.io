JAW (Browser Game)

Jogo multijogador online para navegadores usando HTML5 Canvas para renderizar o jogo e Node.js + Socket.io para o servidor e comunicação. O código está disponível no GitHub: Repositório JAW.

Aplicação online em: https://jawzinho.herokuapp.com/





Afazer (TODOs Web App)

Aplicação web de TODO list, usando Angular e Node.js. Os códigos fonte estão disponíveis no GitHub: Repositório do Front-end e Repositório do Back-end.

Aplicação online em: https://afazer-web.herokuapp.com/
